# Maison Screenshot Shotlist

## Goal

Create screenshot assets that sell calm, shared home coordination, and premium simplicity.

The screenshots should not look like raw app captures with random data. They should feel staged, elegant, and believable.

## Visual direction

- calm and premium
- domestic, not enterprise
- warm but clean
- clear hierarchy
- minimal clutter in mock data

## Copy rules

Use:
- short headlines
- plain language
- calm emotional framing
- outcome-focused copy

Avoid:
- overly technical feature descriptions
- tiny text overlays
- generic productivity jargon
- screenshots overloaded with too many UI details

---

## Asset prep notes

Before capturing screenshots:
- use clean, believable household sample data
- remove debug text and test artifacts
- make sure dates and labels look intentional
- keep names and household members consistent across shots
- choose screens with strong visual hierarchy
