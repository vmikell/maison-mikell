Maison • Visual Mockups

Landing page and dashboard mockups for each Maison color story

Prepared for Victor. These are visual direction mockups, not coded product screens. The goal is to see how each palette behaves on the two surfaces that matter most right now: first impression and daily use.

“Can you make visual mockups of what the dashboard and landing page would look like for each palette?”Yes. This deck shows each color story on both surfaces so the differences are easier to feel, not just describe.

What to look for

Which palette makes Maison feel most like a home brand, which one makes the dashboard easiest to trust, and which one feels distinct enough to carry the identity long term.

My bias

I still think Sunwashed Market is the strongest first test overall, but Clay Hearth gets more interesting once the type gets stronger.

Recommendation

What stands out after seeing them visually

Strongest overall

Sunwashed Market still wins as the most balanced direction. It feels warm and human without losing energy, and it holds together well on both the landing page and the dashboard.

Most trustworthy dashboard

Sage Linen is the easiest to imagine using every day. It feels calm and stable, which is a real advantage for a product people return to constantly.

Strongest brand personality

Clay Hearth becomes really compelling once the typography sharpens up. It has the most soul, but it needs a stronger wordmark and more disciplined UI handling.

Next move I’d recommend

Pick one of these for a tighter second pass, then I’ll turn it into a more refined landing page direction and a real dashboard style system with stronger type, spacing, and component rules.
